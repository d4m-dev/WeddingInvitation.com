## 📦 Tài liệu

* Chạy lệnh `npm install`, sau đó `npm run dev`, và mở `http://localhost:8080`.
* Thay đổi nội dung tệp `index.html` theo ý muốn của bạn.
* Nếu bạn không muốn sử dụng **tính năng bình luận**, hãy xóa các thuộc tính `data-url` và `data-key` trong phần tử `<body>` của index.html.
* Điều chỉnh `data-url` trong phần `<body>` của index và dashboard cho phù hợp với URL backend (nếu bạn tự host).
* Đồng thời điều chỉnh `data-key` trong index với khóa truy cập mà bạn có thể lấy từ bảng điều khiển.
* Nếu muốn sử dụng GIF, hãy lấy khóa API Tenor tại developers.google.com/tenor.
* Để triển khai, hãy chạy `npm run build:public`. Thư mục `public` là thư mục bạn sẽ tải lên.
* Đối với backend tự host, hãy xem giải thích bên dưới, hoặc sử dụng **API dùng thử** miễn phí.

> Thiệp mời này chỉ sử dụng HTML, CSS và JavaScript thông thường. NPM được sử dụng để các tệp JavaScript có thể được thực thi trực tiếp (không còn là loại module nữa).

> Nếu bạn vẫn muốn không dùng NPM, hãy thay đổi `src="./dist/guest.js"` thành `src="./js/guest.js" type="module"` trong thẻ `<head>` của index và dashboard.html, với rủi ro giao diện bị lỗi khi tải lần đầu.

> Nếu bạn có câu hỏi, hãy sử dụng tính năng `discussions` để những người khác cũng có thể đọc được.

> [!WARNING]  
> Hãy sử dụng phiên bản 3.14.0, vì phiên bản 4 vẫn đang trong giai đoạn phát triển và có thể có lỗi 🐛
